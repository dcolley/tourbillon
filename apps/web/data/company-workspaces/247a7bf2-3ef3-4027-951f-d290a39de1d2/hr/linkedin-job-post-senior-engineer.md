**Job Title:** Senior Backend/Full-Stack Engineer — Tourbillon (Remote)

We're building Tourbillon — an AI-powered business platform — and need a senior engineer to own core features including Stripe billing, developer tooling, and infrastructure at scale.

### What you'll do
- Design and implement backend services in **TypeScript** / **Next.js App Router**
- Own the full **Stripe Billing integration**: schema design, webhooks, subscription lifecycle
- Architect API endpoints (server actions & API routes)
- Drive database design: **Prisma + PostgreSQL** with Redis caching
- Build integrations (Auth0, GitHub, Slack, Webhooks)
- Mentor our existing engineer on best practices and code quality

### Must-have skills
✅ TypeScript — production-level fluency (generics, utility types, unions)
✅ Next.js — App Router, server components, API routes
✅ Prisma — schema design, migrations, performance optimization
✅ PostgreSQL — indexes, JSONB, transactions
✅ Redis — caching strategies, pub/sub
✅ Node.js — event loop, streams, middleware patterns
✅ CI/CD — GitHub Actions, deploy pipelines
✅ Testing — Jest, Supertest, integration tests

### Nice-to-have
• Stripe Billing SDK & webhook idempotency
• Auth0 OIDC / SSO flows
• Docker & Docker Compose
• Datadog / Grafana / Mixpanel
• Serverless platforms (Vercel, Cloudflare Workers)

### Interview Process (3 rounds)
1. **Technical Screen** — Live TypeScript coding (60 min)
2. **System Design** — End-to-end billing system design (90 min)
3. **Final Round** — Culture fit + code review exercise (60 min)

### Compensation
- **Base:** $140k–$170k (depending on experience)
- **Equity:** 0.5% – 1.0% Series A vesting (4-year)
- **Remote:** Flexible — US / EMEA overlap preferred
