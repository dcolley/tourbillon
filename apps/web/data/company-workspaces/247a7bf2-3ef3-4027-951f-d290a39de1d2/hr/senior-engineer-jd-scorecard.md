# Senior Backend/Full-Stack Engineer — Job Description & Hiring Scorecard

## Role Summary
Tourbillon is seeking a **Senior Backend/Full-Stack Engineer** to own core platform features including Stripe billing, developer tooling, and infrastructure at scale.

## Responsibilities
- Design and implement backend services (TypeScript, Next.js App Router)
- Own the Stripe Billing integration: schema, webhooks, subscription lifecycle
- Architect and implement API endpoints (Next.js server actions / API routes)
- Drive database schema design and migrations (Prisma, Postgres, Redis)
- Build integrations (Auth0, GitHub, Slack, Webhooks)
- Implement performance monitoring, CI/CD, and infrastructure hardening
- Mentor the existing Engineer on best practices and code quality
- Write clean, tested, and documented code

## Must-Have Skills
- **TypeScript**: Production-level fluency (generics, utility types, unions)
- **Next.js**: App Router, server components, API routes
- **Prisma**: Schema design, migrations, performance (select, includes)
- **Postgres**: Indexes, JSONB, transactions
- **Redis**: Caching strategies, pub/sub
- **Node.js**: Event loop, streams, middleware patterns
- **CI/CD**: GitHub Actions, deploy pipelines
- **Testing**: Jest, Supertest, integration tests

## Nice-to-Have
- Stripe Billing SDK & webhook patterns
- Auth0 OIDC / SSO flows
- Docker & Docker Compose
- Datadog / Grafana / Mixpanel
- FaaS experience (Vercel Serverless / Cloudflare Workers)

## Interview Process (3 Rounds)
1. **Technical Screen (60 min)**: Live coding in TypeScript (e.g., implement a rate limiter or webhook idempotency handler)
2. **System Design (90 min)**: Design Tourbillon's billing system end-to-end (schema → webhook processing → dashboard)
3. **Final Interview (60 min)**: Culture fit, past projects, and a code review exercise
