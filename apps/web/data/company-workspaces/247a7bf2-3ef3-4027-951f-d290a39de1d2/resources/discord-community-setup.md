## 2. Channel Structure (Sets up)
```
📢 ANNOUNCEMENTS
├── #announcements - Official product updates from the team

🗣️ GENERAL
├── #general - Main discussion hub
├── #introductions - New member welcome

👀 SHOWCASE
├── #showcase - User projects & creations

💡 IDEAS & FEATURES
├── #feature-requests - Community-driven ideas

🔧 DEV CORNER
├── #dev-corner - Technical discussions (APIs, integrations, etc.)
├── #help-desk - Q&A and troubleshooting
```

## 3. Roles & Permissions Sets up
### Roles (in order):
1. **Owner** (Admin)
2. **Moderator**
3. **Beta Tester** (auto-assigned on join)
4. **Community Member** (after 3 days)
5. **Power User** (after first post in #showcase)
6. **Visitor** (default)

### Role Colors:
- **Beta Tester**: `#00BCF2` (Blue)
- **Community Member**: `#43B581` (Green)
- **Power User**: `#FEE75C` (Yellow)
- **Visitor**: `#95A5A6` (Grey)

## 4. Onboarding Flow
### Step 1: Auto-Roles
- Discord permalinks for self-assignment
- Assign "Beta Tester" role automatically via welcome channel: `/welcome/here`
- Use Discord's built-in "Welcome Screen"

### Step 2: Welcome Message (Pinned in #introductions)
```
🎉 Welcome to the Tourbillon Community!

This is the official Tourbillon Discord server for early adopters, beta testers, and AI enthusiasts.

What is Tourbillon?
Tourbillon is an open-source platform that lets you design multi-agent workflows, automate complex processes, and launch AI-powered systems in minutes — not months.

👋 Introduce yourself!
Share your name, role, and what you're building with Tourbillon.
```

### Step 3: Server Rules (Pinned in #general)
```
📜 Tourbillon Server Rules
1. Be respectful and constructive
2. Use #help-desk for questions before creating new channels
3. Tag appropriately: `@Beta Tester` for major announcements
4. Default: `#general` for First, Reserved for important updates
5. Chat with intent: No idle chit-chat beyond the first 24 hours.
6. Privacy: Share your data is yours. Tourbillon is built with privacy-by-design.

For feature requests, head to #feature-requests and use the format:
**Feature:** [Name]
**Problem it solves:** [Brief description]
**Desired Outcome:** [What users should be able to do]
```
