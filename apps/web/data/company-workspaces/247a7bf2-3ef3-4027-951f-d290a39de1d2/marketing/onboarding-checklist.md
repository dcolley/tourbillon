# Tourbillon Customer Onboarding Checklist (Non-Technical)

---

## Welcome Journey: First 30 Minutes

### Step 1 — Sign Up & Access Your Account
- [ ] Visit tourbillon.io and click **Sign Up**
- [ ] Create your account with email, or use Google/GitHub single sign-on
- [ ] Verify your email address via the confirmation link sent to your inbox
- [ ] Complete your profile (name, role, avatar)

### Step 2 — Explore the Dashboard
- [ ] Log in and navigate to your **Dashboard** home screen
- [ ] Familiarize yourself with the layout: Goals panel, Active Projects, Agent Activity feed
- [ ] Click on any existing sample project or template to see how it's structured
- [ ] Hover over each sidebar item to see tooltips describing its function

### Step 3 — Set Up Your Workspace
- [ ] **Create your first Team**: Go to Settings → Team & Members → New Team
- [ ] **Invite teammates**: Share an invite link or enter their emails directly (no setup required on their end)
- [ ] **Assign roles**: Admin, Editor, or Viewer — keep it simple for now
- [ ] Verify that at least one team member has accepted the invitation

---

## First 15 Minutes: Your First Goal & Project

### Step 4 — Create Your First Goal
- [ ] Navigate to **Goals** from the left sidebar → Click **New Goal**
- [ ] Write a clear, concise goal statement (e.g., *"Automate our weekly report generation"*)
- [ ] Add relevant tags or categories for organization
- [ ] Set a target date if applicable

### Step 5 — Create Your First Project Under That Goal
- [ ] From your new Goal page → Click **New Project**
- [ ] Give the project a descriptive name (e.g., *"Weekly Report Automation"*)
- [ ] Add a brief description of what the project is trying to achieve
- [ ] Save and confirm the Goal ↔ Project linkage appears correctly

### Step 6 — Create Your First Task / Subtask
- [ ] Inside your new project → Click **Add Task**
- [ ] Write a simple task description in plain language (e.g., *"Fetch last week's data from our analytics dashboard"*)
- [ ] Save the subtask and observe how it appears in the project view

---

## First 15 Minutes: Meet Your Agents

### Step 7 — Assign an Agent to Execute Your Task
- [ ] Click into your newly created task
- [ ] Look for **Assign Agent** or **Let AI Help** button on the right panel
- [ ] Select an available agent (or let Tourbillon auto-select)
- [ ] Watch as the agent begins planning and executing — observe the real-time progress feed

### Step 8 — Review Results & Provide Feedback
- [ ] Monitor the task execution in the **Agent Activity** view
- [ ] When complete, review the delivered output
- [ ] Click **Approve** if satisfied, or provide feedback with a comment for revision
- [ ] Celebrate 🎉 — your first AI-executed task is done!

---

## First Hour: Level Up

### Step 9 — Explore Advanced Features (Optional but Recommended)
- [ ] **Agent Configuration**: Define custom prompts and tools for agents under Settings → Agents
- [ ] **Project Templates**: Save a project structure as a template for reuse (click ⋯ → Save as Template)
- [ ] **Integrations**: Connect any tools you use daily — Slack, GitHub, Notion, Google Drive

### Step 10 — Set Up Notifications & Preferences
- [ ] Go to Settings → Notifications and customize how/when you want alerts:
  - Task complete notifications
  - Agent error / failure warnings
  - Weekly project summary reports
- [ ] Enable or disable email digests based on your preference

---

## First Week: Establish Rhythm

### Step 11 — Build a Recurring Workflow
- [ ] Identify one repetitive task you do regularly (reporting, data collection, status updates)
- [ ] Create a Goal → Project → Task chain for it
- [ ] Save as a template so you can reuse it every time
- [ ] Try running the agent on it and compare output quality to your manual process

### Step 12 — Check In & Iterate
- [ ] Review all tasks completed this week in your **Dashboard**
- [ ] Note which agents handled what types of work well
- [ ] Leave feedback on any task that didn't go as expected
- [ ] Adjust your goal/project structure based on learnings

### Step 13 — Join the Community & Get Support
- [ ] Visit our **Community Forum** or **Discord** (link in footer)
- [ ] Introduce yourself and share one thing you automated this week
- [ ] Bookmark the **Help Center / Docs** for quick reference
- [ ] Follow Tourbillon on social channels for product update announcements
