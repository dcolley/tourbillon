## Product & Value

### Q: What is Tourbillon?
**A:** Tourbillon is an **AI agent orchestration platform** that empowers teams to create, configure, and deploy autonomous AI agents. Agents plan work, access tools/APIs, collaborate across workspaces, and support human-in-the-loop workflows. The platform structures work as Goals → Projects → Tasks, providing visibility and automation at scale.

### Q: How does Tourbillon solve enterprise pain points?
**A:** Enterprises face siloed teams, fragmented tools, and manual handoffs. Tourbillon solves this with:
1. **Visibility:** Goal → Project → Task hierarchy for cross-functional tracking.
2. **Automation:** AI agents that plan, execute, and iterate—reducing manual workflows.
3. **Governance:** Enterprise security (SSO, RBAC, SOC2-ready), audit trails, and compliance reporting.

## Security & Compliance

### Q: What security practices does Tourbillon follow?
**A:** Tourbillon follows industry-standard practices:
- **Encryption in transit:** TLS/SSL.
- **Secure authentication:** OAuth, SSO, RBAC.
- **Regular security reviews and vulnerability assessments.**

### Q: What data does Tourbillon collect?
**A:** Tourbillon collects:
- Account Information (name, email via OAuth or manual entry).
- Usage Data (Agent configurations, workflows, feature metrics).
- Device Information (IP address, browser type, OS, access times).
- Analytics (cookies for understanding user behavior).
All data is encrypted in transit (TLS/SSL) and you retain ownership of content created through the Service.

### Q: How does Tourbillon handle user data?
**A:** Tourbillon follows industry-standard practices:
- Encryption in transit (TLS/SSL) and at rest.
- Secure authentication (OAuth, SSO, RBAC).
- Regular security reviews and vulnerability assessments.
- Access controls restrict employee data access.
