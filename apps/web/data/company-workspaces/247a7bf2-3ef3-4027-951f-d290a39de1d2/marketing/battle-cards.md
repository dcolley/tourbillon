## Low-code Automations (Zapier / Make)

| Attribute | Zapier/Make | Tourbillon | Winner |
|-----------|------------|------------|--------|
| Target | SMBs, solo-founders | Enterprise AI Teams | **Tourbillon** |
| Core Value | Single-step or multi-step visual workflows | Goal-driven AI agent orchestration | **Tourbillon** |
| Pricing | Per task/scenario ($20–$60/mo) | Per agent ($29/mo/agent) | **Tourbillon** |
| AI | Add-on, manual updates | Native AI agents that plan, execute, iterate | **Tourbillon** |

### Why Switch
1. **AI-Divergence:** "Make connects apps — Tourbillon connects *intelligence*." Tourbillon agents *think* and *execute*.
2. **Scale:** "Zapier requires 2000+ manual connections → Tourbillon agents access *any API* via a single endpoint. No manual connector needed.
3. **Governance & Compliance:** Tourbillon has native human-in-the-loop workflows, auto-approval, and audit trails built-in.
4. **Speed to Value:** "We need to scale" → Tourbillon agents can be cloned, configured, and deployed in seconds.

### When to Fight
- **One-off with <5 workflows"** → Tourbillon's agent overhead might be overkill.
