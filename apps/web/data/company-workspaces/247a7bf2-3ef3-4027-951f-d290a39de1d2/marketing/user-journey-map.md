## 1. Discovery Channels

How our first 1000 users find Tourbillon:

| Channel | Weight | Description | Est. CAC |
|---------|--------|-------------|----------|
| **Product Hunt Launch** | 30% | Primary launch day push; 300–500 initial upvotes from community | $15–25 |
| **SEO / Organic** | 25% | "Multi-agent orchestration," "no-code AI workflows" keywords | $20–35 |
| **Paid Social (LinkedIn, Twitter)** | 20% | Targeted PMs, Product Managers, Ops teams | $35–50 |
| **Community (Discord, Reddit, Hacker News)** | 15% | Early adopters, developer communities | $10–20 |
| **Partnerships (Press, Blogger, Webinar)** | 10% | Co-hosted webinars, guest posts on partner blogs | $25–40 |

**Discovery Funnel Math (5-stage pipeline)**

- **Impressions → Clicks:** 3% average (7.5% for Product Hunt launch week)
- **Clicks → Signups:** 30% of visitors create a free account
- **Signup → First Goal:** 60% of users complete their first goal
- **First Goal → First Workflow:** 70% of users build their first automated workflow
- **First Workflow → Team Invite:** 40% of users invite at least 1 teammate

## 2. Onboarding Journey (T=0 to T+14 Days)

### Phase 1: Sign-Up (T+0 to T+5 min)
**Goal:** Frictionless account creation → First Goal created

1. Landing page → Click "Start Free"
2. Auth: Google OAuth or email + password
3. Email verification: Confirmation email with deep link
4. Profile setup: Name, role, avatar (optional 30s)

**Trigger:** Immediate welcome email (Email 1)
**Success metric:** ≥85% complete profile within 5 minutes

### Phase 2: First Goal (T+5 min to T+15 min)**
**Goal:** Define first goal → Create first project → Add first task

1. Dashboard tour — 3-step interactive overlay
   - Step 1: "What are you working on? → Goal creation"
   - Step 2: "Where does this fit? → Project creation"
   - Step 3: "What's the first task? → Agent assignment"

**Success metric:** ≥60% of users create Goal → Project → Task within first session
**Trigger:** Auto-assign AI agent to first task; push notification on completion

### Phase 3: Team Activation (Day 1 → Day 7)**
**Goal:** Invite team members → Collaborate on shared project

1. Invite teammates — Single invite link with 3 roles (Admin, Editor, Viewer)
2. First team member joins → Reactivation bonus ("Free agent for 7 days")
3. Collaborate — Shared project view, comments, agent assignments

**Success metric:** ≥40% of solo users invite ≥1 teammate within 3 days
**Trigger:** Email 2 (T+48h): "3 features you're not using yet"

### Phase 4: Habit Formation (Day 7 → Day 21)**
**Goal:** Build regular behavior → Create recurring task → Connect integrations

1. Recurring task creation — Identify at least one repetitive workflow
2. Template creation — Save a project structure for reuse
3. Integrations — Connect 1+ tools (Slack, GitHub, Notion)

**Success metric:** ≥35% of users remain active at Day 7
**Trigger:** Email 3 (T+14 days): NPS survey

## 4. Retention Levers

### Primary Retention Levers
1. **Product-Led Growth** — Free tier hooks with 5-agent limit; Pro $49/mo; Enterprise custom
2. **Community Engagement** — Discord community, Community Council, Weekly AMAs
3. **Win-back Sequence** — 3-email sequence for users inactive 14+ days
4. **Referral Program** — "Give 2 weeks of Pro" for every referred user
5. **Score-Based Engagement:
   - ≤6: Auto-create support ticket, human outreach within 1 hour
   - 7-8: Send "How did we do?" follow-up email
   - ≥9: Trigger referral program, testimonial request

### Retention by Tier
| Tier | Day 30 | Day 60 | Day 90 |
|------|----

|------|

|

| **Free** | 35% | 25% | 20% |
| **Pro** | 60% | 45% | 35% |
| **Enterprise** | 95% | 90% | 85% |

### Churn Prevention Triggers
| Churn Signal | Action | Timeline |
|--------------|--------|----------|
| Login drops ≥60% in 14 days | CS check-in email | Day 15 |
| Payment failure + no login for 7 days | Win-back email sequence | Day 7 |
| Negative NPS (≤6) | CS call + CTO encouragement note | Day 8 |
| Competitor mentions in feedback | Competitive analysis review → tailored value prop outreach | Day 30 |
| Seat utilization <30% | Offer seat reduction or suggest upgrading to higher tier | Day 45 |
