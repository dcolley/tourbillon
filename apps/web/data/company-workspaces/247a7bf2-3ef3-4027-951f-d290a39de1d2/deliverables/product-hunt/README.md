## How to Convert SVG → PNG/JPG

### Option 1: Browser (easiest)
```bash
# Open in browser, use screenshot tool or Print Screen
open deliverables/product-hunt/cover/cover-image.svg
```

### Option 2: ImageMagick (if available)
```bash
convert cover/cover-image.svg[0] -density 300 cover/cover-image.png
```

### Option 3: Online Converter
Upload SVG to [cloudconvert.com](https://cloudconvert.com/svg-to-png) or similar.

### Option 4: Figma / Illustrator
Import SVG directly, export as PNG/JPG at required resolution.
