# Derek Colley — Content Voice & Formatting Requirements

**For all @derekcolley_ content creation (threads, tweets, posts)**

---

## 🚫 CRITICAL RULES (Must Follow)

### 1. ALWAYS write in FIRST PERSON as Derek
- ❌ "Derek already posts about scaling" → ✅ "I've been posting about scaling for months"
- ❌ "Derek's June 29 tweet resonated" → ✅ "My June 29 tweet hit home because..."
- ❌ Any third-person reference to "Derek" → rewrite as "I/my/me"

### 2. HASHTAGS: At most ONE, usually NONE
- Derek rarely uses hashtags on X
- If using any: #LocalAI only (if relevant)
- Never use multiple tags like #AgenticAI #AgentDesign #DGXSpark #OpenSource etc.

---

## 🗣️ VOICE CHARACTERISTICS

### Style (from @derekcolley_ actual posts):
- **Lowercase-heavy** in casual long-form content: "sitting on my dgx spark running..."
- **Direct, blunt observations**: "My AI reports:" not "The system indicates:"
- **Self-deprecating humor**: "sit with that", asking for a friend about hardware
- **Specific numbers woven naturally**: tok/s counts, dollar amounts, VRAM sizes
- **No corporate/marketing language** — no "synergy", "paradigm shift", etc.

### Tone:
- British English
- Casual but knowledgeable (not academic, not salesy)
- Slightly contrarian on topics like pricing models, open vs closed AI
- Honest about tradeoffs and failures ("this crashed my setup because...")
